# 🚗 Vehicle Parking App- NowPark

A web-based **NowPark App** built using Flask. This app allows users to book and release parking spots in various parking lots, while admins can manage lots, spots, and view detailed dashboards.

---

## ✨ Features

### 👥 User Functionalities
- Register and log in
- Book available parking spots automatically
- Release booked spots
- View current parking status
- View reservation history and cost summary

### 🛠️ Admin Functionalities
- Add, update, delete parking lots and spots
- View and manage all users and their bookings
- Access system-wide dashboards:
  - Parking usage stats
  - Cost and revenue per lot
  - User activity reports

---

## 🧰 Tech Stack

- **Backend:** Flask (Python)
- **Frontend:** HTML, CSS, Bootstrap, Jinja2
- **Database:** SQLite
- **ORM:** SQLAlchemy
- **Authentication:** Flask-Login

## ⚙️ Installation Steps

### 1. Clone the Repository
```bash
git clone https://github.com/24f1000503/vehicle-parking-app.git
```

### 2. Navigate to the Project Directory
```bash
cd vehicle-parking-app
```

### 3. Create and Activate Virtual Environment

#### Create:
```bash
python -m venv venv
```

#### Activate:
- **Linux/macOS:**
  ```bash
  source venv/bin/activate
  ```
- **Windows:**
  ```bash
  venv\Scripts\activate
  ```

### 4. Install Dependencies
```bash
pip install -r requirements.txt
```

### 5. Run the App
```bash
python run.py
```
